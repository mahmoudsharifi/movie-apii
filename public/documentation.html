This API provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Request Method URL Response
Get all movies GET /movies List of movies
Get one movie GET /movie A movie
Register new user POST /user "User has signed up"
Change username PUT /user "Your username is now ___"
Delete user DELETE /user "Your acount has been deleted"
Add movie to favorites POST /favorite "___ has been added to your favorites"
Remove movie from favorites DELETE /favorite "___ has been removed from your favorites"
